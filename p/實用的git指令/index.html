實用的Git指令

一些進階的、實用性高的指令

Jan 06, 2023
1 minute read

前言

整理一下工作上有用過，並覺得實用、比較少人提過的一些Git指令。

Git指令

  1. 退回到上一個Commit，數字可以更改，1就是上一次，改成三的話就是三次前的Commit
1

git checkout HEAD~1

  1. 退回上一次的Commit，重複輸入的話會往復循環
1

git checkout -

  1. 將本次修改合併至上次Commit，如果有漏掉的提交會常用這個指令
1

git commit --amend --no-edit
